# 关于我

我是昇哥。

我是一名前端出身、后来一路走向全栈的开发者。  
写代码、做产品、想架构，是我这些年一直在做的事。

但我不是学院里的研究者，也不想把自己写成什么“高人”或“老师”。  
更准确地说，我只是一个很普通的人。普通本科，没有特别光鲜的履历，走过一些弯路，也挨过一些现实的打击。

这些年，对我影响很深的，不是某一次“成功”，而是几次让人停下来的时刻。

母亲病危进 ICU 的时候，我第一次那么直接地感受到，很多事情并不是理所当然。  
后来工作上遇到被拖欠工资、劳动仲裁、公司突然暴雷倒闭。  
再后来，左手骨折，原本想做的很多事被迫按下暂停。

这些事情没有让我一下子明白很多大道理。  
它们只是让我慢慢安静下来，开始认真问自己：

- 我到底想过怎样的生活？
- 我真正想做的事是什么？
- 我为什么而学，为什么而工作，为什么还要继续往前走？

也是从那时起，我开始更认真地接触《易经》，接触道家的文本。

一开始并不是为了研究，也不是为了显得特别。  
只是因为在读这些东西的时候，我会慢慢静下来。  
它们会让我重新去看“变化”“关系”“时机”“处境”这些过去常常被我忽略的问题。

我为什么学《易经》？

因为它对我来说，不只是一本古书。  
它更像一种训练。训练我如何看变化，如何看势，如何看自己身在何处。  
它不能替我生活，也不能替我做决定，  
但它会让我在很多时候停下来，重新看清眼前这一局。

我为什么要做 `Dao`？

因为我不想把这些学习，只停留在“我看过”“我感兴趣”“我被触动过”。  
对我来说，整理本身就是学习。  
很多东西如果只是读过，当下觉得有感觉，过些日子就散了。  
但当我开始重述、做索引、写下自己的理解，再回头去看，它们才会一点点沉淀下来，变成真正属于自己的东西。

我也知道，像我这样的人并不少。  
很多人对传统文化、对“道”、对《易经》有兴趣，但常常卡在第一步：不知道从哪里开始，不知道先读什么，也不知道怎么把古书和今天的生活连起来。

所以我希望 `Dao` 先成为一个入口。  
它不是百科全书，不是结论中心，也不是一个故作高深的展示页。  
它更像是一口慢慢往下打的井：我一边学习，一边整理，一边把自己走过的路，尽量留成后来者也能看懂的入口。

我做这个项目，不想太功利。  
至少在现在，我更在意的不是结果，而是过程。  
认真读一章，认真写一页，认真想明白一个词，认真记下一个此刻的体会。  
这些事都很小，但我相信，很多真正的变化，本来就是这样一点点发生的。

我也还在路上。  
所以这里不会是一个高高在上的答案站，而更像是一个持续生长的学习仓、整理场和共学入口。

如果你来到这里，也正好对这些内容感兴趣，  
那欢迎一起慢慢读，慢慢看，慢慢长出自己的理解。
