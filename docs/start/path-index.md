# 路径与别名索引

这页的作用很直接：

- 让人知道每篇内容的正式路径
- 让协作者知道该用什么 slug 命名
- 让后续搜索、RAG、评论系统挂载有稳定锚点

## 当前规则

- 页面访问路径统一使用英文 slug
- 页面标题保留中文
- 每个文档至少保留 2-5 个别名
- 新增文档时，先登记路径，再开始写正文

## 新增文档前先问

1. 它属于哪个板块？
2. 它属于哪个模块？
3. 它的英文 slug 是什么？
4. 它的中文标题是什么？
5. 它的常用别名是什么？

先答完，再建文档。
