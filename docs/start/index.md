# 站点导览

这个站不是“大而全”的传统文化百科，也不是只写个人感悟的博客。

它当前更像一个入口：

- 为初学者建立进入路径
- 为持续学习者建立索引与结构
- 为作者自己的长期研究建立文档骨架

这里默认只展示“给读者看的内容”。

开发计划、里程碑、贡献日志、问题模板等内部材料，不放在站点主导航里，而统一维护在仓库的 `dev/` 与 `.github/`。

## 你可以怎么使用这个站

### 如果你刚开始接触道家或易经

先看：

- [学习计划](/roadmap/study-plan)
- [道家总览](/daoism/)
- [易经总览](/yijing/)

### 如果你已经在读原典

优先看：

- 各原典导读页
- 术语、专题与阅读地图
- 作者的阶段性解读

### 如果你是协作者

优先看：

- `/AGENTS.md`
- [路径与别名索引](/start/path-index)
- [内容模型](/standards/content-model)
- [RAG 约束](/standards/rag-guidelines)
