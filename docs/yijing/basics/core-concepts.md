# 核心概念

当前优先建立这些概念卡：

- 阴 / 阳
- 刚 / 柔
- 中 / 正
- 位
- 应
- 时
- 变易
- 象 / 理 / 用

每个概念建议后续单独成页，而不是全部堆在一篇大文里。

原因：

- 更利于检索
- 更利于内链
- 更利于后续做知识库切片
