# 学习方法

你当前按卦序学习，这是很好的主线。

关键不是改掉它，而是给它加两条辅线：

## 主线：按卦序推进

- 保持连续感
- 保持原典接触
- 保持长期节奏

## 辅线一：概念专题化

在学卦的同时，把反复出现的概念抽出来：

- 阴阳
- 刚柔
- 中正
- 应
- 变易

## 辅线二：结构化记录

每学一个卦，都至少留下一个小节点：

- 卦名
- 卦象
- 核心句
- 我的三问
- 可关联的专题

这样做的好处是：

- 不打断主线学习
- 又能逐步长出可搜索、可关联、可复用的知识网络
