## 这个站在做什么

`Dao` 先是一个导读型资料库。

它帮助读者：

- 知道从哪里开始读
- 知道不同原典之间的关系
- 知道一个概念应该落在哪个语境中理解
- 知道作者自己的学习与解读路径

## 推荐进入顺序

1. 先看 [站点导览](/start/)
2. 再看 [学习计划](/roadmap/study-plan)
3. 然后进入 [道家](/daoism/) 或 [易经](/yijing/)

## 站点当前原则

- 先结构，后堆内容
- 先导读，后百科
- 先重述，后扩写
- 先沉淀，后长出更深的理论
