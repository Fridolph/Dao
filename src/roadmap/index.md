# 阅读地图

`Dao` 当前推荐的路径不是“全量通读”，而是“先建立地图，再持续深入”。

## 第一阶段：建立方向感

- 道家与易经分别是什么
- 它们有哪些交汇，又有哪些差异
- 现代人为什么值得重新进入这些文本

## 第二阶段：建立稳定阅读路径

- 先读可进入的导读
- 再读原典
- 再回到术语、专题与笔记做交叉理解

## 第三阶段：形成自己的问题意识

- 我为什么想学
- 我想用它理解什么
- 我在当代生活与工作中如何重新理解这些概念
