## 这是什么

**Dao** 是一个以传统文化、道学、易经为入口的 **导读型资料库**，核心任务只有一件：

> 帮人找到进入「道 / 易经 / 传统文化」的入口，并提供可追踪的阅读路径。

它不是百科、不是论文仓、也不是现代图书的搬运站。当前阶段为 **首版导读型资料库建设期**。

## 站点结构

```
首页 ─┬─ 开始（站点导览 / 路径索引）
      ├─ 阅读地图（学习计划 / 推荐书单）
      ├─ 道家
      │   ├─ 入门三经（清静经 / 阴符经 / 太上感应篇）
      │   ├─ 道德经（81 章）
      │   ├─ 庄子（33 篇）
      │   ├─ 列子（8 篇）
      │   └─ 道教经典（阴骘文 / 坐忘论 / 天隐子）
      ├─ 易经
      │   ├─ 基础概念（8 篇）
      │   ├─ 学习方法
      │   └─ 六十四卦（卦辞 / 彖传 / 象传 / 爻辞）
      ├─ 专题（阴阳 / 无为 / 变易 / 象理用）
      ├─ 我的解读（个人笔记与理解）
      └─ 关于（站点缘起 / 关于作者）
```

## 内容分层原则

项目始终按三层归位内容：

1. **公共知识层** — 原典导读、阅读顺序、术语解释、学习地图（面向初学者）
2. **个人解读层** — 三问解读、学习笔记、专题文章（带有作者视角）
3. **Dao 理论层** — 成熟方法论与核心命题（暂未开始，需更高沉淀度后开放）
