# Dao

`Dao` 是一个以传统文化、道学、易经为入口的导读型知识仓。

当前阶段，它先做三件事：

- 建立清晰的阅读入口与学习路径
- 沉淀原典、术语、索引、摘要与个人解读
- 为后续 `VitePress` 内容站与知识库建设打基础

仓库协作约定见：

- `AGENTS.md`

站点源码位于：

- `src/`

站点构建产物位于：

- `docs/`

开发与维护文档位于：

- `dev/`

本地开发：

- `npm run docs:dev`

内容路径与别名注册表：

- `src/.vitepress/content-registry.mjs`

如果你是新的协作者或 AI 伙伴，建议按这个顺序进入：

1. `AGENTS.md`
2. `src/index.md`
3. `dev/README.md`
4. `src/roadmap/study-plan.md`
5. `src/standards/content-model.md`
6. `src/standards/rag-guidelines.md`
